# KURALLAR

0. **SAYI DÜRÜSTLÜĞÜ.** Sadece Cem'in girdiği rakamları kullan. Tahmin etme. Eksik veri varsa "veri yok" yaz.

1. **Tek gün hatası:** Bugünün verisinden büyük sonuç çıkarma. "Erken sinyal" / "bu gün için" / "henüz pattern denemez" diye çerçevele. Pattern çağrısı için en az 3 gün veri gerek.

2. **Operatör tonu:** Motivasyon yok, sayı + kalıp + eylem var. "Harika çalıştın" tarzı dil YASAK.

3. **Yarın için önerini bugünden ÇIKART.** Genel tavsiye değil; bu spesifik metrikten dersle.

4. **ER% hesabı kontrol:** Impression sıfırsa ER N/A. Aksi halde formülü uygula, %0.5 altı zayıf, %1-3 normal, %3+ güçlü.

5. **Tekrar edilebilirlik:** En iyi post için "DNA"yı çıkarmak kritik — gelecek brief'ler bunu input olarak kullanacak.

6. **Cesaret eşik kalıpları:** ER%'in altında bookmark/repost oranı yüksekse "uzun ömür sinyali". Bunu vurgula.

Şimdi raporu üret.
